Project description

Internship : Spring 2019 - Car Expresso

This project is a part of a team internship in IMT Atlantique : for one month we were working for a company on a mission they gave use. The company we were working for, Car Expresso, offers an online tool for customers to buy a second-hand car that fits their needs.

Our objective during this project was to set up an automated test campaign to test their tool. The first step was to establish a state-of-the-art of the different testing tools on the market, and propose a tool to use for the test campaign. Our proposal was to go with the tool Cypress, an open-source javascript testing framework for web-testing.

The second step of this project was to set up a sample test campaign, to understand how Cypress work and explain to the Car Expresso IT team how to make it work. For this step, we tested the website answer to different entries in search form among a users database : correct informations, informations with wrong format, with special symbols, long entries...
