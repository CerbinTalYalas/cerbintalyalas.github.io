TeLoCiDeSi

Project description

Project Leader : Louis DURET-ROBERT

TeLoCiDeSi stands for Ternary Logic Circuits Designer and Simulator. As the name stands, this software is used to design and simulate the behaviour of logic circuits using ternary logic.

This project was conducted by a team of 6 students from IMT Atlantique, France, based on Louis Duret-Robert's personnal work. It relies on Python and Tkinter library to propose an user-friendly and intuitive application for ternary logic.

Telocidesi gives the user the tools to design, save and edit their own ternary circuits along with a basic documentation on ternary architecture. The software also comes with an excution part, allowing to run a simulation and check the state of the circuit inputs/outputs on a chronogram.
